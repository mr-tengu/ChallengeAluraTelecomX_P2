# 🤖 Predicción de Churn - Telecom X (Parte 2)

Este repositorio contiene la fase de Machine Learning para la detección temprana de abandono de clientes en Telecom X.

## 🚀 Objetivos del Proyecto
* Preparar los datos tratados para modelos de aprendizaje automático.
* Entrenar un modelo predictivo para identificar clientes en riesgo.
* Determinar los factores clave (Feature Importance) que impulsan la evasión.

## 📊 Resultados del Modelo
* **Accuracy:** ~79%
* **Variables Clave:** El tipo de contrato, los cargos mensuales y la antigüedad del cliente son los factores determinantes.

## 🛠️ Herramientas
* Python & Pandas para manipulación.
* Scikit-Learn para el modelo de Random Forest.
* Seaborn & Matplotlib para el análisis de importancia.

## 💡 Recomendación de Negocio
Se sugiere implementar una estrategia de "Lealtad para Nuevos Clientes" que ofrezca descuentos en contratos de 1 año para aquellos que actualmente están en modalidad mes a mes, reduciendo así la volatilidad inicial.
